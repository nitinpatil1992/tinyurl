# TinyURL

Short url generator with capacity to habdle over 1 Billion urls (13!)


# Execution

```
$ docker-compose build 
$ docker-compose up
# wait for apps to load
```

Visit [localhost:9000/tinyurl](http://localhost:9000/tinyurl) to use tinyurl webservice
